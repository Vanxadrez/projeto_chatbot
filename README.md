# 🎤 Assistente de Voz com IA (Whisper + Gemini)

## 📌 Sobre o Projeto

Este projeto implementa um assistente de voz em Python utilizando:

- Gravação de áudio via JavaScript no Google Colab
- Transcrição de fala com Whisper (OpenAI)
- Geração de resposta com Gemini (Google AI)
- Conversão de texto para voz utilizando gTTS

O fluxo completo funciona da seguinte forma:

1. O usuário grava um áudio
2. O áudio é convertido para texto
3. O texto é enviado para um modelo LLM (Gemini)
4. A resposta é convertida novamente em áudio

---

## 🛠 Tecnologias Utilizadas

- Python
- Google Colab
- OpenAI Whisper
- Google Gemini API
- gTTS
- JavaScript (para captura de áudio)
